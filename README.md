# ediop3Ai
An AI inside python


## setup

```
git clone https://github.com/ediop3SquadALT/ediop3Ai
cd ediop3Ai
unzip ediop3Ai
cd ediop3Ai
```
for Linux
```
sudo bash setup.sh
```
for termux
```
bash setup.sh
```

finally run it


Linux (kali / parrot os)
```
sudo python3 ediop3Ai.py
```

for termux
```
python3 ediop3Ai.py
```
